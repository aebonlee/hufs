# HUFS 과정 개편 — 4시간 동일 과정 2개 캠퍼스 진행 (2026-07-02)

## 변경 배경
기존: "대학 행정 실무를 위한 바이브 코딩" · 8시간(4시간 × 2회, 1회차+2회차 순차).
발주처 요청으로 **동일 4시간 과정을 2개 캠퍼스에서 각각 1회씩** 진행하는 구조로 변경.
(원안은 Claude Code·Claude Design·VS Code+Claude Code 연동 등을 충분히 나눠 진행하려 했으나, 캠퍼스 2곳 4시간 동일 진행으로 확정됨)

- **과정명**: 대학 행정 실무를 위한 바이브 코딩 **기초**

- **구성**: 1부 10:00~12:00 (2교시) · 2부 13:00~15:00 (2교시)
  - 교시 통합: 기존 1·2교시 → 1교시, 3·4교시 → 2교시 (각 60분)
- **일정·장소** (동일 내용 2회)
  - 7/28(화) 서울캠퍼스 교수학습개발원 1층 108호 (PC 실습실)
  - 7/30(목) 글로벌캠퍼스 백년관 2층 213호 (멀티미디어교육실)

## 코드 변경 요약
- `src/config/site.ts` · `index.html` · About · RecommendedSites: 과정명(기초), "8시간(4시간×2회)" → "4시간(2개 캠퍼스 동일 진행)"
