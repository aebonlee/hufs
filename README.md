# HUFS Vibe Coding

한국외국어대학교(HUFS) 교직원 대상 **대학 행정 실무를 위한 바이브 코딩** 교육 사이트.
Claude 기반, 8시간(4시간 × 2회) 과정.

- 배포: https://hufs.dreamitbiz.com
- 1회차: Claude 바이브코딩 입문 & 행정 업무 자동화
- 2회차: 행정 실무 웹도구 제작 & 배포

## 개발

```bash
npm install
npm run dev      # 개발 서버 (http://localhost:5174)
npm run build    # 프로덕션 빌드
npm run deploy   # gh-pages 배포
```

React 19 · Vite 7 · TypeScript · Supabase · React Router 7
